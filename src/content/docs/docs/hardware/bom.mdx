## Pre-assembled kits

If you prefer not to source parts individually:

- **Wowrobo** — worldwide shipping, ~$579 for a full kit
- **Taobao** — China, ~¥3,699

Wowrobo also sells pre-assembled SO-101 arms separately (~1.5 hours saved on arm assembly).
